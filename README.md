## üöÄ Getting Started

### Prerequisites

- PHP 7.0 or higher
- Web server (Apache, Nginx) or PHP's built-in server

### Installation

1. Clone this repository:
```bash
git clone <repository-url>
cd proxy
```

2. Start the PHP development server:
```bash
php -S localhost:8000
```

3. Open your browser and navigate to:
```
http://localhost:8000/proxyscrapper.php
```

## üìñ Usage

### Proxy Scraper

1. Select the proxy type (HTTP, HTTPS, SOCKS4, SOCKS5, or ALL)
2. Enter the number of proxies you want to scrape
3. Click "Start Scraping"
4. Wait for the process to complete
5. Download the `proxies.txt` file

### Proxy Checker

1. Open `Proxychecker.php` in your browser
2. Upload or paste your proxy list
3. Click "Check Proxies"
4. View results with response times and status
